CARA MENGGUNAKAN

### Cloning repositori
```
git clone https://github.com/muchammadwr/submission.git
```
### Membuka Repository
```
cd submission
```

### Instalasi library 
```
pip install streamlit
pip install -r requirements.txt
```

### Membuka file

```
cd dashboard
```

### menjalankan streamlit

```
streamlit run dashboard.py
```
